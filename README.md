# Short-lol-quest
Проект по дисциплине Основы алгоритмизации и программирование.

Игра short lol quest это не обычный шутер или сюжетка, это игра в которой можно расслабиться, ведь она длится около 5 минут, немного жуткой атмосферы для побега из комнаты.

Целевая: подойдет для всех возолстов от 3+
